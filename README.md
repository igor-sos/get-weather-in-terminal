# Узнать погоду в терминале

Простая программа для получения погоды в терминале для нескольких городов.

## Описание проекта

Скрипт выводит актуальную погоду в терминале для заданных локаций (Лондон, аэропорт Шереметьево, Череповец).  
Данные берутся с сервиса [wttr.in](https://wttr.in) — без регистрации и API-ключей.

## Как установить

1. Убедитесь, что у вас установлен [Python 3.14.6](https://www.python.org/downloads/release/python-3146/).
2. Склонируйте репозиторий:

```bash
git clone https://github.com/igor-sos/get-weather-in-terminal.git
```

```bash
cd ./get-weather-in-terminal
```

Установите зависимости:

```Bash
pip install -r requirements.txt
```

## Как запустить:

```Bash 
python main.py
```
